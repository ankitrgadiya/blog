Github Student Developer Pack

27 May 2016

Last night I was going through my mails and there was one from Github. They accepted my request for Student Developer Pack. This pack include a lot of freebies and free tools, those which can be used by students to create an online presence, and start developing cool stuff.

This pack includes:
Tools Description
Atom A hackable text editor for the 21st century
Open source by Github, free for everyone
AWS Educate Access to the AWS cloud, free training and collaboration resources
$ 15 bonus AWS credits for a total of $50-$115
Bitnami Install cloud applications in a single click
Business 3 plan free for one year
CrowdFlower Crowdsourcing and data enrichment platform
Access to the Crowdflower platform
Digital Ocean Simple cloud hosting, built for developers
$ 50 in platform credit for new users
DNSimple Simple DNS management with one click serves and a robust API
Personal hosted DNS plan for one year
Github Powerful collaboration, code review, and code management
Unlimited private repositories while you are a student
Hackhands Live programming help available 24/7
$25 in platform credit
Microsoft Azure A suite of Microsoft Azure cloud services and developer tools, including the VIsual Studio IDE
Microsoft Azure, Visual Studio Community 2015, and rest of the Microsoft Developer tools while you're a student
Namecheap Affordable registration, hosting, and domain management
One year domain name registration on the .me TLD
Send Grid Email infrastrucutre as a service
Student plan 15K emails/month while you're a student
Stripe Web and mobile payments, build for developers
Waived transaction fees on first $1000 in revenue processed
Thinkful Learn Fundamentals of Web Development with a mentor
Two week course in web development that will get you started coding. Includes two 1-on-1 mentor sessions, unlimited office hours, and membership in the Thinkful community
Transifex Localization platform that easily integrates with your code base
One year free of the Starter Plan, a $99/month value. get 50,000 hosted words, unlimited projects, and access to translation partners to bring your software to a global market from the start.
Travis CI Continuous integration platform for open source and private projects
Private builds while you're a student.
Udacity Enroll in a Nanodegree program, and launch your career in Web and Mobile Development, Machine Learning, Data Science, and more
One month free access to any Nanodegree program
Unreal Engine A complete suite of game development tools for PC, console, mobile, web and VR
Unreal Engine while you're a student

Really cool huh?? If you're a student developer and want some or all of these you can go here.
